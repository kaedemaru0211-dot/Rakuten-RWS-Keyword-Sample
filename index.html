RAKUTEN WEB SERVICE

Rakuten RWS Keyword Sample

楽天市場商品検索APIを使って、keyword検索で商品情報を取得する最小サンプル。 価格比較と仕入れ候補抽出の検証用であり、スクレイピングは含まない。

アプリ名: Rakuten RWS Keyword Sample
運営者名: Hisaki Oura
連絡先: kaedemaru0211@gmail.com
公開URL想定: https://kaedemaru0211-dot.github.io/Rakuten-RWS-Keyword-Sample/

アプリ名

  • Rakuten RWS Keyword Sample

用途

  • keyword で楽天市場の商品を検索
  • itemName, itemPrice, itemUrl, shopName を取得
  • 価格比較候補の抽出を検証

入力

  • keyword
  • minPrice
  • maxPrice
  • availability
  • hits

出力

  • itemName
  • itemPrice
  • itemUrl
  • shopName

データ取得方針

  • 楽天市場の公式APIを利用
  • スクレイピングは行わない
  • APIキーは `.env` で管理

実行例

python scripts/search_keyword.py "Braun 替刃 52B" --min-price 3000 --max-price 12000 --hits 10

出典

商品情報は Rakuten Web Service の楽天市場商品検索APIを通じて取得する。 このページは API 申請時のアプリ説明ページを兼ねる。

詳細は Rakuten Ichiba Item Search API を参照。
